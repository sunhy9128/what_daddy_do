# 爸爸去哪了

**一款给准爸爸和新手爸爸的孕产期全程陪伴助手。**

从备孕到产后，让每一位准爸爸不错过任何重要时刻，陪伴妈妈安心度过每一天。

[![Platform](https://img.shields.io/badge/Platform-iOS%20%7C%20Android%20%7C%20Web-blue)](#)

---

## 设计的初心

老婆怀孕的那一刻，我既兴奋又慌张。

兴奋的是我们要当爸妈了。慌张的是——我该做些什么？

网上查的、群里问的、医院发的……信息多且杂，我记不住。产检时间、准备物品、注意事项，每次都临时抱佛脚。作为爸爸，我能做的事好像不多，但，我不想只是"陪伴"。

我想要**真正参与**这件事。

所以我做了「爸爸去哪了」——一个帮助准爸爸从备孕到产后全程参与的助手 app。它会告诉你每个阶段该做什么、该准备什么，让你不那么慌，不那么迷茫。

**陪她，更是在陪她一起迎接新生命。**

---

## 解决什么问题

| 场景 | 没有 App | 有「爸爸去哪了」 |
|------|---------|-----------------|
| 不知道下次产检是什么时候 | 翻聊天记录 / 问她 | 首页直接显示倒计时 |
| 不知道该准备什么待产包 | 网上搜、手忙脚乱 | 各阶段任务清单清晰列出 |
| 宝宝出生后喂养不规律 | 记不住几点喂的奶 | 喂养计时器帮你记录 |
| 不知道疫苗什么时候打 | 翻疫苗本 / 问邻居 | 疫苗日历自动提醒 |
| 错过宝宝成长的重要时刻 | 事后才知道 | 阶段里程碑自动庆祝（还有彩纸特效 🎉）|

## 覆盖五大阶段

从备孕到产后，每一个重要时刻都有我们陪伴。

| 阶段 | 说明 |
|------|------|
| 🌱 备孕期 | 提前准备，优生优育 |
| 🤰 孕早期 | 1-12 周，安心度过不稳定期 |
| 👼 孕中期 | 13-27 周，成长关键期 |
| 🎈 孕晚期 | 28-40 周，准备迎接新生命 |
| 🍼 产后 | 新生儿期，共同成长 |

## 核心功能

### 📋 智能任务提醒
基于孕周智能生成每周任务，产检、营养、准备事项不再遗漏。

### 📅 产检倒计时
提前提醒产检时间，输入预产期自动计算当前孕周，再也不会错过任何重要检查。

### 🍼 喂养计时器
新手爸妈最常问的问题：上次喂奶是什么时候？喂养计时器帮你记住。

### 📈 生长图表
根据 WHO 标准生长曲线，在家就能记录和观察宝宝的身高体重发育情况。

### 💉 疫苗追踪
输入疫苗接种计划，日历视图清晰展示，带宝宝打疫苗不再靠提醒。

### 🎉 里程碑庆祝
从备孕到产后，每个新阶段到来时送上祝贺页面和彩纸动画。**你不是在旁观，你在参与。**

### 📖 知识社区
精选的孕育知识文章，已读状态自动同步。与其他爸妈交流经验，互相鼓励。

## 开始使用

```bash
# 安装依赖
npm install

# 启动开发服务器
npm start

# 指定平台运行
npm run ios      # iOS
npm run android  # Android
npm run web      # Web
```
